### virtualenv
```
$ pip install virtualenv --user

$ mkdir venom
$ virtualenv -p /usr/bin/python3 venom
$ cd venom/

$ source  bin/activate

$ pip install requests
(copied under lib/python3.6/site-packages/requests)

$ deactivate

```
